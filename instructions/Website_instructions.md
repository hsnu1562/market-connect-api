# Website Instructions

This is a intruction for web structure.

```
EO:Event Organizer
```

## Client Side

### Welcoming Page

**Function**: Display the logo and allow users to select whether they are a Merchant or an Event Organizer.

![Welcoming Page Layout Mockup](/market-connect-api/instructions%20_asset/Website_instructions_picture/welcome%20page.png)

### Merchant Main Page

**Function**: A homepage for merchant to search venue. This page should have the majority function of searching venue.

### EO Main Page

**Function**: A homepage for event organizers to monitor venue bookings. It provides an at-a-glance view of unpaid bookings, paid bookings, and estimated revenue.

### Login Page

**Function**: Log in and link to member center.
*For login: Email, Password, Captcha*

### Sign up Page

**Function**: Sign up. Merchant and EO all should sign up and choose their character here.
*For Sign up: Charater, Real Name, Email, Phone Number, Password, Comfirm Password, other information(to be added)*

### Member Center Page

**Function**: Display the members personal information, appointment history, payment history, and site information (OG only).

### Site Information Page

**Function**: Display the venue details. The information should be uploaded by the EO.
*This template applies to all venue details; every venue will have its own unique webpage.*

### Site Information Upload Page

**Function**: EO should type their site information here.

### Payment Page

**Function**: Merchant should choose their pay method and pay here.
